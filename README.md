# Academic Grade Management System

## Giới thiệu
Academic Grade Management System là một ứng dụng dòng lệnh (CLI) được phát triển bằng Java, áp dụng nguyên lý lập trình hướng đối tượng (OOP). 

Hệ thống cho phép người dùng nhập dữ liệu, truy xuất thông tin từ cơ sở dữ liệu SQLite, thực hiện tính toán điểm số và trả về kết quả đã được định dạng.

---

## Tính năng chính
- Nhập dữ liệu từ người dùng qua Command Line Interface (CLI)
- Kết nối và truy xuất dữ liệu từ cơ sở dữ liệu SQLite
- Tính điểm môn học, điểm tổng kết 
- Hỗ trợ nhiều loại bài tập và cách tính điểm khác nhau
- Hiển thị kết quả rõ ràng cho người dùng

---

## Công nghệ sử dụng
- Ngôn ngữ: Java  
- Cơ sở dữ liệu: SQLite  
- Kiến trúc: Object-Oriented Programming (OOP)  
- Giao diện: Command Line Interface (CLI)  

---

## Chức năng chính của hệ thống
- Quản lý thông tin sinh viên  
- Xử lý dữ liệu điểm từ database  
- Tính toán điểm của học sinh
- Hiển thị kết quả cho người dùng
